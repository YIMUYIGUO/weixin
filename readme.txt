智能化条材套裁优化、精确材料重量查询和基于WordPress的幕墙知识管理工具

== Description ==

材料计算助手是一个专业的建筑材料计算工具，完全集成WordPress内容管理系统，为工程师和建筑专业人员提供以下功能：

**主要功能：**

* **条材套裁优化** - 使用智能算法计算最优切割方案，支持锯缝宽度设置，最大化材料利用率
* **材料重量查询** - 精确查询各种型材理论重量，涵盖角钢、工字钢、槽钢等常用建筑材料
* **幕墙知识库** - 完全基于WordPress文章系统的专业幕墙技术知识管理

**WordPress集成特色：**

* 使用WordPress原生文章系统管理幕墙知识
* 支持分类和标签系统进行内容组织
* 特色图片支持，增强视觉效果
* 完整的搜索和筛选功能
* 管理员可灵活标记文章为幕墙知识内容
* 支持文章的编辑、发布、删除等完整生命周期管理

**技术特色：**

* 智能优化算法，减少材料浪费
* 国标数据计算，确保精确性
* 响应式设计，完美支持移动端
* 用户记录保存，方便历史查看
* React组件化开发，用户体验流畅
* WordPress AJAX集成，无缝数据交互

== Installation ==

1. 上传插件文件到 `/wp-content/plugins/material-calculator/` 目录
2. 在WordPress管理后台的"插件"页面激活插件
3. 插件会自动创建必要的数据库表和示例内容
4. 插件会自动创建幕墙知识相关的分类
5. 在页面或文章中使用短代码来显示相应功能

== 短代码使用 ==

* `[material_calculator_home]` - 显示工具首页，包含所有功能入口
* `[cutting_optimizer]` - 条材套裁优化工具
* `[weight_calculator]` - 材料重量查询工具
* `[curtain_wall_knowledge]` - 幕墙知识库（基于WordPress文章）

== 幕墙知识管理 ==

### 管理员操作：

1. 进入WordPress管理后台 > 材料计算 > 幕墙知识
2. 查看所有已发布的文章
3. 选择要显示在知识库中的文章
4. 点击"标记为幕墙知识"
5. 被标记的文章将在前端幕墙知识功能中显示

### 内容创建建议：

* 为幕墙知识文章添加适当的分类（如：防雷系统、转接件等）
* 使用标签来标记技术要点
* 添加特色图片增强视觉效果
* 内容应包含专业的幕墙技术知识
* 使用Markdown格式编写结构化内容

### 自动创建的分类：

插件激活时会自动创建以下幕墙知识分类：
- 安装节点
- 防雷系统  
- 转接件
- 密封胶
- 结构胶
- 五金配件
- 玻璃选型
- 铝型材
- 石材幕墙
- 金属幕墙
- 玻璃幕墙
- 质量控制
- 安全施工
- 维护保养

== Frequently Asked Questions ==

= 如何添加新的幕墙知识内容？ =

1. 在WordPress后台创建新文章
2. 选择合适的分类和标签
3. 添加特色图片（可选）
4. 发布文章
5. 在"材料计算 > 幕墙知识"中标记为幕墙知识

= 如何修改已有的幕墙知识内容？ =

直接在WordPress后台编辑对应的文章即可，修改会实时反映在前端。

= 计算结果是否准确？ =

重量计算基于国标数据，套裁优化使用经过验证的算法，但实际应用中请结合具体情况进行调整。

= 是否支持自定义样式？ =

插件提供了完整的CSS类，您可以通过主题的自定义CSS来调整样式。

= 数据存储在哪里？ =

幕墙知识内容存储在WordPress原生的文章表中，套裁计算记录存储在插件创建的专用数据表中。
